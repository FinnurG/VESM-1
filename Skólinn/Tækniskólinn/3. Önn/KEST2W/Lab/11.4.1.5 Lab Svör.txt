Question:
Hvað gera þessi command

CD: sýnir current directory

DEL: eyðir eins og nafnið segir

DIR: sýnir directory af userinnum

DISKPART: Opnar nýtt cmd til að managea disk og disk partitions og volume

EXIT: Lokar cmdinu

FORMAT: formattar diskinn í diskpart cm

GPRESULT: setur reglur á hluti

MD: make directory býr til directory

TASKLIST: sýnir task list og hversu mikið memory það notar og aðrar uppýsingar

RD: removes directory öfugt um 

ROBOCOPY: sama og XCOPY en bara fleiri settings

SHUTDOWN: er hægt að gera timer á shutdown og meira

XCOPY: copyar hluti
